About

Exodia OS is a Stunning & Powerful Linux distribution, Which is based on Arch Linux.

Exodia OS is built for wireless PenTesting.
Exodia comes with 3 Editions : Home, Predator, Wireless.
Exodia comes with BSPWM as the default Window Manager, 13+ fancy Themes for BSPWM That make Exodia look very awesome.
Exodia has a special edition for Acer Predator Laptops with predator sense CLI pre-installed to control RGB backlight and fan speed.
Exodia will support Tools for all cybersecurity tracks, All window managers and desktop environments.
